# React + Tailwind (Frontend)

Proyecto frontend con React (Vite) y Tailwind CSS.

## Qué incluye

- Vite con React
- Tailwind CSS + PostCSS
- Estructura básica (`src/main.jsx`, `src/App.jsx`, `src/index.css`)

## Requisitos

- Node.js (v16+ recomendado)
- npm o yarn

## Instalar y ejecutar (PowerShell)

```powershell
cd c:\Users\luis9\Pantalla_figma\react-tailwind
npm install
npm run dev
```

Abre http://localhost:5173 en tu navegador.

## Construir para producción

```powershell
npm run build
npm run preview
```


Si quieres, puedo también inicializar el repositorio git desde aquí y hacer el primer commit (necesitaré permiso para ejecutar comandos).
